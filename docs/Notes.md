# Notes Index

The old monolithic task list was split on 2026-04-10 after a repo audit.

- [notes/README.md](notes/README.md): audit rules and status legend.
- [notes/todo.md](notes/todo.md): tasks that are still open or only partially addressed.

Legacy task numbers are not globally unique. Use the section plus the task number together when referencing an item.
