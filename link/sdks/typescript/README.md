# KalamDB TypeScript SDKs

This folder contains the publishable TypeScript packages:

- `client/` for `@kalamdb/client`
- `consumer/` for `@kalamdb/consumer`
- `orm/` for `@kalamdb/orm` (Drizzle ORM driver and schema generator)

Use each package directory as the source of truth for its build, test, and publish workflow.
